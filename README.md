# Digital Service Management System (DSMS)

A console-based **Digital Service Management System** developed in **C Language** as a Fundamental of Programming (FOP) semester project.

The application provides multiple digital services through a menu-driven interface, allowing users to access different online services from a single platform.

---

## Features

### 📚 Online Course Registration
- View available courses
- Display instructor details
- Calculate monthly course fee
- Select course duration
- Display total course cost

### 🚗 Car Rental Service
- Budget, Standard, and Luxury categories
- View available cars
- Calculate rental charges
- Weekly discount support

### 🏨 Hotel Reservation System
- Hotels available in multiple Pakistani cities
- Room reservation
- Night-wise billing
- Hotel availability management

### ✈️ Ticket Booking System
#### Flight Booking
- Select airline
- Passenger information
- Seat quantity
- Fare calculation

#### Train Booking
- Display train information
- Ticket availability
- Ticket reservation

### 📢 Digital Marketing Advertisement
- Advertisement for:
  - Shop
  - Business
  - Company
- Platform selection
  - Facebook
  - Instagram
  - YouTube
- Advertisement duration
- Pricing based on follower count
